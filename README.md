# prometheus-ai

Monitoring and alerting

## Part of Luminous-MastermindAI Ecosystem

## Installation

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Security and Dependency Governance

```bash
# Run direct dependency N/N-1 compliance check
pnpm run security:n-1

# Run vulnerability advisory audit
pnpm run security:audit

# Run both checks
pnpm run security:all
```
